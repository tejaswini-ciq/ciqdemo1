Import Graphical User Interface (GUI) from Excel Example

This example project provides the example Excel based specification of a graphical user interface in Shopping Application UI.xls in the example project folder. The importer speeds up and can even enable creation of models for test generation without the need to draw structure diagrams by creating screens, forms, button and hyperlinks based on the GUI specification provided in Excel. For more information about Excel based GUI specification or using this importer please see in the Conformiq Manual section Creating Models with Conformiq Creator > Using Conformiq Creator > Importing Graphical User Interfaces from Excel.

To import the sample Excel file right click on the example project in the Project Explorer view and select Import ... > Import Graphical User Interface from Excel, select the Shopping Application UI.xls file from the Creator project folder and click the Finish button. Expand and inspect the structure diagram generated into the model folder in the Project Explorer view and browse the created interface objects. Right click on the canvas and select Arrange Diagram to improve the layout. Use quick fixes offered in the Problems view to add layout information to the diagrams.

After that simply create a new activity diagram by right clicking on the model folder and selecting New > Activity Diagram, and start specification of, e.g., login functionality, by using the actions created from the imported GUI information in activity node specification!
